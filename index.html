About me

I FIRST GOT INTO THE TECHNOLOGY WORLD AROUND THE AGE OF 10 WHEN I RECIEVED MY FIRST COMPUTER. WITH THAT ALONE, I WAS ABLE TO TEACH MYSELF THE BASIC FUNCTIONS OF HOW COMPUTER SCIENCE WORKED.

I TAUGHT MYSELF THE IN'S AND OUTS OF CS BY REVERSE ENGINEERING DIFFERENT MACHINES AND WEBSITES. SOFTWARE REVERSE ENGINEERING (SRE). FROM THAT ALONE, I FIGURED OUT HOW SOFTWARE CODE, AND OF COURSE, FRONT / BACK END WEB CODE WORKED. FROM THERE, I WAS ABLE TO CODE AN ENTIRE WEBSITE FROM SCRATCH.

MANY OF THE WEBSITES I BUILD ARE HANDMADE FROM SCRATCH AND REVOLVE AROUND THE IDEA OF SIMPLICITY. I USE HTML, CSS (OFTEN BOOTSTRAP), AND JAVASCRIPT IF NEEDED. WHEN COMBINED, IT CREATES BEAUTIFUL, ANIMATED, AND MODERN WEBSITES SUCH AS THIS ONE.

I HAVE EXTENSIVE KNOWLEDGE ABOUT HASHING, WIRELESS ATTACKS, AND DATA ANALYSIS BY USING THIRD-PARTY INSTALLED LINUX SYSTEMS VIA KALI LINUX COMMANDS AND THEIR APPS. ALONG WITH THAT, I HAVE FURTHER EXPERIENCE IN HOW THE DARK WEB WORKS, INCLUDING .ONION DOMAINS AND HOW TOR'S HIDDEN SERVICES WORK.

BESIDES WEB CODE, I'VE RECENTLY STARTED GETTING INTO NEW HOBBIES, SUCH AS LEARNING HOW TO CODE DISCORD BOTS (BLOB), BUILDING CUSTOM-MADE HARDWARE COMPUTERS FROM SCRATCH (WINDOWS), UX DESIGN, AND RECENTLY, TRYING TO LEARN ABOUT HUMAN PSYCHOLOGY.
